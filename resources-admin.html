Add Resource

Create a new ICT resource for booking allocation.

Collection-only resources do not require a deployment location and will not appear on the DE Dashboard.
Select the accessories that can physically work with this specific device resource. This will become the source of truth for booking-page filtering and allocator matching.

Edit Resource

Update resource details, availability, installed software and lifecycle actions.

Changing the device / accessory type or resource grouping will generate a new Resource ID so the ID remains consistent with the updated resource classification.
Collection-only resources do not require a deployment location and will not appear on the DE Dashboard.
Select the accessories compatible with this specific device resource. Leave blank if no accessories should be offered with this resource.
Resource lifecycle actions Use delete only when the resource should no longer appear as a bookable item. If future bookings exist, the backend will retire the resource instead.

Add Software

Manage specialised software that can be assigned to resources.

Retired software remains visible in historical resource records but will not appear as a selectable option when assigning software to resources.

Software lifecycle actions If this software is assigned to any resources, it will be retired instead of permanently deleted.

Booking Import

Preview and import legacy booking records so they can appear in My Bookings and the Deployment Dashboard.

Paste rows copied from Excel, or upload a CSV file. The first row must contain column headers such as Booking Date, Start Time, End Time, Requester Email, Location, Device Type, Number of Devices, and optional accessory quantity columns such as Mouse Qty, Headset Qty, or Apple Pencil Qty.

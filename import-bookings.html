Booking Import

Preview and import legacy booking records so they can appear in My Bookings and the Deployment Dashboard.

Upload an Excel (.xlsx) or CSV file, or paste rows copied from Excel. The importer will detect the legacy report header row automatically. Supported headers include Booking Date, Start Time, End Time, Requester Email, Location, Device Type, Number of Devices, and optional accessory quantity columns such as Mouse Qty, Headset Qty, or Apple Pencil Qty.
Supports Excel workbook (.xlsx), CSV (.csv), text (.txt), or pasted rows copied from Excel.
Only rows with booking dates within this range will be previewed and committed. Leave both dates blank to process the full file.
